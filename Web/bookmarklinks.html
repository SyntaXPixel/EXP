Introduction to HTML

HTML stands for HyperText Markup Language
HTML uses elements represented by tags, such as <p> for paragraphs and <a> for links
HTML documents are structured in a tree-like format with a root element
HTML is the backbone of web pages and is often used in conjunction with CSS and JavaScript
HTML introduced new features such as semantic elements, audio, video support, and improved form
